# product-slot-management

A Python-based simulation of an Amazon fulfillment center conveyor belt using arrays. This project demonstrates slot access, product search, slot updates, and belt capacity checking while showcasing basic data structure operations.

# gps_navigation_system

A Python implementation of a GPS Navigation System using two stacks. The program simulates browser-like back and forward navigation, allowing users to visit new locations, return to previous checkpoints, and move forward when available.

# toll_plaza_simulation

A Python implementation of a Toll Plaza Simulation using a Circular Queue. The project demonstrates enqueue, dequeue, queue display, and efficient reuse of empty slots in a fixed-size queue.
